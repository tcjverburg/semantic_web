Pasta Ontology

This page lets you query my own ontology I created for the third assignment. As you have probably inferred, this was a pasta ontology. For purpose of simplicity, I have two queries set ready on this page which you can query against a stardog triple store.

However, both query windows on this page are quite different: The first one has inferencing/reasoning turned on whilst the second window has inferencing/reasoning turned off. The difference this makes is quite evident after pressing the buttons. This is because the specific query that I have designed for my particular ontology yields no results without inferencing. Yet, because of certain properties I assigned to various instances in the pasta ontology, it does yield results if the inferencing is turned on. This is because I query for certain properties of which I know that are only inferred in my ontology.
